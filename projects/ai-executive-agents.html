AI Executive Agents

Built a suite of 13 AI agent tools for the COO, Deputy CFO, and wider executive team, enabling natural-language access to governed financial and operational data across Snowflake's semantic layer.

Snowflake Cortex Semantic Views Python SQL Sigma

Business Problem

Senior executives — including the COO, Deputy CFO, and wider leadership team — needed rapid access to complex financial and operational data but were dependent on analyst teams to pull reports. They required the ability to ask questions in natural language and receive accurate, governed answers within seconds — not days.

Why It Was Difficult

  • Data spanned multiple domains: finance, commercial, product, and customer
  • Accuracy was non-negotiable — executives make decisions worth millions on these numbers
  • Existing dashboards answered pre-defined questions only; ad-hoc analysis still required human intervention
  • Semantic consistency across 246 governed KPIs had to be maintained

My Role

I owned the end-to-end design and delivery of the AI agent system — from architecture through to deployment and evaluation. This included defining the semantic layer, designing the tool architecture, building the evaluation framework, and presenting to the executive team.

Approach

Commercial

  • Identified the highest-value questions the CFO's office asked repeatedly
  • Quantified analyst time saved per question type
  • Prioritised tools by business impact and data readiness

Technical

  • Built 13 specialised tools covering revenue, retention, acquisition, product performance, forecasting, and cohort analysis
  • Designed a semantic layer in Snowflake ensuring all metrics had single, governed definitions
  • Implemented guardrails to prevent hallucination and ensure answers traced back to source data

Evaluation

  • Created a multi-dimensional evaluation framework testing accuracy, relevance, and completeness
  • Ran blind tests comparing agent answers to analyst-produced answers
  • Iterated on prompt engineering and tool design based on evaluation results

Architecture

Executive (Natural Language Question)
        ↓
   Cortex Agent
        ↓
   Tool Selection (13 tools)
        ↓
   Semantic Views (governed metrics)
        ↓
   Snowflake (source data)
        ↓
   Formatted Response → Executive

Technologies

  • Snowflake Cortex — Agent orchestration and LLM inference
  • Semantic Views — Governed metric definitions
  • Python — Tool logic and evaluation framework
  • SQL — Complex analytical queries
  • Sigma — Visual verification and dashboard cross-reference

Outcome

  • Executive team able to self-serve complex analytical questions
  • Analyst time freed for higher-value strategic work
  • Response time reduced from days to seconds for routine questions
  • Framework established for scaling AI agents across other business functions

Lessons Learned

  • Semantic governance is the foundation — without clean, trusted metric definitions, AI agents produce confident but wrong answers
  • Evaluation must be continuous, not a one-off exercise
  • Executive trust is earned through transparency: showing the source data behind every answer is critical
  • Start with narrow, high-confidence tools rather than one broad agent that hallucinates
